# Commands

This page explains the commands available in EasyTeamShuffle.

EasyTeamShuffle creates teams from members currently connected to a voice channel.

---

## Team Shuffle

Create teams from members in the current voice channel.

### Japanese Command

```
/チーム分け
```

### English Command

```
/teams
```

You can also specify the number of teams.

Example:

```
/teams 3
```

This will create **3 teams** using the members in the voice channel.

---

## Re-shuffle

After teams are created, a **Re-shuffle button** appears in the result message.

Pressing this button will reshuffle the teams without running the command again.

To prevent spam, the button has a **10 second cooldown** after use.

---

## Requirements

To use EasyTeamShuffle commands:

* You must be connected to a **voice channel**
* The bot must have permission to **send messages and embeds**
